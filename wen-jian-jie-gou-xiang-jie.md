# 文件简介
linux下一切皆文件，包括内存，磁盘，网络都可以用文件的形式表示。操作系统的主要功能之一就是文件管理，所以文件是一种非常重要的资源。目录是一种特殊的文件，他的存储文件的名称和相关信息。


# 文件组成
文件主要由两部分组成，一是文件的元信息，由inode表示，二是文件的数据，由block表示


# inode
用于存储文件的各属性，包括

- 所有者信息：
    - owner
    - group
    - other


- 权限信息：
    - read
    - write
    - execute


- 时间信息
    - 修改时间（mtime）
    - 访问时间（atime）
    - 状态时间（ctime）
    

- 标志信息
    - 一些flags


- 内容信息
    - type
    - size
    - 对应的block的位置信息


# block
用来存储文件的内容数据
